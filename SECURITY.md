# Security Policy

## Supported Versions

| Repository | Supported |
|---|---|
| cortex | Current release |
| ViralForge | Current release |

## Reporting a Vulnerability

If you discover a security vulnerability in any TECHKNOWMAD LABS project:

1. **Do NOT** open a public issue
2. Email **admin@techknowmad.ai** with:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Assessment**: Within 5 business days
- **Fix**: Dependent on severity, targeting 30 days for critical issues

## Scope

This policy applies to all repositories under the TECHKNOWMAD-LABS organization.

## Recognition

We appreciate responsible disclosure and will credit reporters (with permission) in release notes.
